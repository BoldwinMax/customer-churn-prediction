# custormer-churn-prediction-
This project analyzes customer data to predict customer churn using machine learning models. The goal is to identify customers who are likely to leave so that businesses can take proactive measures.

## Project Overview

- **Dataset**: Customer demographics, account details, and service usage data.
- **Objective**: Predict customer churn and evaluate model performance using metrics like Recall and ROC-AUC.
- **Tools Used**: Python, Pandas, NumPy, scikit-learn, XGBoost, Matplotlib, Seaborn, Jupyter Notebook.
